VScodeでPtyhonを使えるようにする for Windows

python Step1
投稿日 2025年01月31日

pythonのver.確認とver.更新

  • https://kamesuke-blog.com/programming/python_version/ ※ver.更新の部分は記事の通りにできなかったため、下記の通り色々触りながら更新しました。イマイチ良く分からないまま更新できてしまった節もあるので、上手くいかなかった場合は申し訳ありません。
  • ダウンロードしたpythonどこにあんねん？Userフォルダーの中にAppDataのファイル無いんだが？となった人は隠しファイルを表示する設定にする https://www.jaskun.com/windows10/win10-appdata/

ver.更新

画面右下のver.をクリックする。画像では「3.11.9」。 ※.pyファイルを開いている時しか右下にver.が表示されません。
インタープリターの選択という項目が画面中央上部に開くので、「インタープリターパスを入力」を選択する
「検索…ファイルシステムを参照してPythonインタープリターを検索します」を選択する
Pythonファイルをインストールしているところまで行って、python.exeファイルを選択する
選択したpythonのverを選択できるようになっているはず
python 3.9.13 (anaconda)があるのは、以前Udemyを受講した際にanacondaを入れていたから
